# SDG-13-Climate-Action-Website
A collaborative web development project building an interactive website to raise awareness about SDG 13 (Climate Action).Developed the Action Impact Simulator (AIS) page — a fully interactive tool featuring 6 themed action cards with dynamic impact scoring (Low/Medium/High feedback levels). Implemented JavaScript selection logic and dynamic background switching based on user-selected impact level. Designed and coded all embedded CSS for card layout, selected/unselected states, and responsive styling. Built the Splash Screen (splash.html) for the website's landing experience. Collaborated with a team to deliver a cohesive, functional multi-page website
